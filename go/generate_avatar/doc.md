# 头像生成 Generate Avatar

根据输入内容生成像素风格图片，可用作默认头像

生成png图片，长宽比为1:1，整体上左右对称；图片主体为紧密排列的5*5方格（色块），外围有一圈白边

- 图片长宽像素数为`12a*12a`，其中`a`为用户输入的**尺寸等级**
    - 每一个小方格边长为`2a`，外围的白边宽度为`a`
    - 举个例子，假设尺寸等级为4，则最终图片像素数为`48*48`
- 每个方格显示指定颜色或白色
    - 指定颜色：根据输入字符串计算，详见实现小节
- 为什么外围有一圈白边：可以方便剪裁为圆形而保留基本样式

## 实现

1. 输入一个**字符串**和想要的图片**尺寸等级**，根据输入字符串计算hash
    - hash algorithm：`sha256`
    - 结果类型：16进制字符串（长度：64）
2. `64 = 15 * 4 + 4`：将hash分为16组，前15组表示对应位置是否显示色块，最后一组值用来表示图像颜色
    - 15组信息用来表示左侧两列与中间列的方格是否显示色块，图片左右对称
    - 一个小方格显示色块的条件：该组最后一个bit为1
        - 每组4位16进制字符、16个bit，如果最后一个bit为1则显示色块、为0则显示白块
        - 每组的前15个bit实际上没有用到
    - 色块的颜色：假设最后一组hash为`0xabcd`，则色块颜色为：`RGBA(ab, bc, cd, da)`
3. 记录以上计算结果，使用`image`包编码图片信息，序列化并写入文件
